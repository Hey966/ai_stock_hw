AI 股票分析與即時通知系統（手機版）
作者：郭昶邑
日期：2026/03/25

==================================================
一、專案簡介
==================================================
本專案使用 Python 建立一個 AI 股票分析系統，
整合即時股價、技術分析（MA20）、AI判斷、
MQTT 即時推播與 Web Dashboard。

系統可在 Android 手機（Termux）上完整運行，
達到類似智慧投資助理的效果。

==================================================
二、系統功能
==================================================
1. 即時抓取台股資料（Yahoo Finance）
2. 計算技術指標（MA20）
3. 計算漲跌百分比
4. AI 投資建議（BUY / SELL / HOLD）
5. MQTT 即時推播通知
6. Android 手機通知（Termux）
7. Web Dashboard 視覺化顯示
8. JSON API（/stocks）

==================================================
三、系統架構
==================================================
auto_stock_mqtt.py
    ↓（抓股價 + AI分析）
MQTT Broker（broker.hivemq.com）
    ↓
mqtt_receiver.py（手機通知）
    ↓
app.py（網頁儀表板）

==================================================
四、檔案說明
==================================================
1. auto_stock_mqtt.py
   - 抓取真實股票資料
   - 計算 MA20、漲跌%
   - 產生 BUY / SELL / HOLD
   - AI 分析建議
   - 發送 MQTT
   - 輸出 latest_results.json

2. mqtt_receiver.py
   - 接收 MQTT 訊息
   - 顯示手機通知
   - 顯示價格、漲跌%、AI建議

3. app.py
   - Web Dashboard（網頁介面）
   - 顯示股票分析結果
   - 自動刷新（每60秒）
   - 提供 /stocks API

4. latest_results.json
   - 最新股票分析資料
   - 提供網頁使用

==================================================
五、安裝環境
==================================================
1. 安裝 Termux（建議 F-Droid）
2. 安裝 Termux:API App
3. 更新系統

pkg update && pkg upgrade -y

4. 安裝套件

pkg install python -y
pkg install git -y
pkg install nano -y
pkg install termux-api -y

pip install paho-mqtt requests flask

5. 開啟儲存空間權限

termux-setup-storage

==================================================
六、資料夾位置
==================================================
/storage/emulated/0/ai_stock_hw

==================================================
七、執行方式
==================================================

【1】進入專案資料夾

cd /storage/emulated/0/ai_stock_hw

--------------------------------------------------

【2】啟動 MQTT 接收（手機通知）

python mqtt_receiver.py

--------------------------------------------------

【3】啟動股票分析系統

python auto_stock_mqtt.py

功能：
- 每60秒抓取股價
- 計算 MA20
- 計算漲跌%
- 發送通知

--------------------------------------------------

【4】啟動 Web Dashboard

python app.py

開啟瀏覽器：

http://127.0.0.1:5000

API：

http://127.0.0.1:5000/stocks

==================================================
八、分析邏輯
==================================================
1. 計算 MA20（近20日平均）
2. 計算漲跌%

判斷：

- 價格 < MA20 且 > 合理價 → SELL
- 價格 < 合理價 → BUY
- 其他 → HOLD

==================================================
九、AI 判斷
==================================================
系統根據價格、MA20、合理價，
產生自然語言投資建議：

- BUY → 建議分批布局
- SELL → 建議減碼
- HOLD → 建議觀望
